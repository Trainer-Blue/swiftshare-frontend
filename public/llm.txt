# SwiftShare.in - LLM Context

## Overview
SwiftShare.in is a real-time collaborative text editor that enables seamless document sharing and live collaboration. Built with modern web technologies, it provides a premium minimalist experience for teams and individuals.

## Key Features
- **Real-Time Collaboration**: Multiple users can edit documents simultaneously with conflict-free synchronization powered by Yjs CRDTs
- **Instant Sharing**: Share documents via unique URLs - no sign-up required
- **Live Sync**: See edits appear in real-time across all connected devices
- **Dark Mode**: Full dark theme support with persistent preferences
- **Connection Status**: Visual indicators for connection state and automatic reconnection
- **Rich Text Editing**: Powered by Quill editor for a smooth writing experience
- **Premium Design**: Minimalist glassmorphism UI for distraction-free editing

## Technology Stack
- Frontend: React + Vite
- Backend: Node.js + WebSockets
- CRDT Engine: Yjs for conflict-free replication
- Database: MongoDB for document persistence
- Real-time: WebSocket-based synchronization
